Хоча це і досить суб'єктивна думка автомобіль Bugatti Type 35 розглядається багатьма як один з найкрасивіших довоєнних гоночних автомобілів легендарної компанії Bugatti. Його краса може зрівнятися тільки з його досягненнями. Це один з найуспішніших автомобілів. Він здобув перемогу в більш ніж тисячі гонках, ставши володарем Світової серії Гран-прі в 1926 році з 351 перемогами. Протягом двох років він також встановив 47 рекордів. З 1925 по 1929 Bugatti Type 35 домінував у гонці Targa Florio.

Перша поява

Перший Bugatti Type 35 був представлений 3 серпня 1924 року. Він був оснащений модифікованим двигуном, який використовувався Type 29. Трехклапанный рядний восьмициліндровий дволітровий двигун з верхнім розподільним валом був оснащений п'ятьма головними підшипниками і володів потужністю близько 90 кінських сил. Підвіска складалася з листових ресор, закріплених на суцільних осях. Гальмівне зусилля вироблялося барабанними гальмами на задній осі. Вони управлялися тросовими приводами, які можна було побачити з зовнішньої частини автомобіля. У загальній складності було вироблено 96 автомобілів цієї моделі.

Існувало кілька версій Type 35, які були спеціально розроблені для розміщення різних типів гонщиків. Type 35A, який отримав прізвисько Tecla, був недорогий версією Type 35. Вперше він був представлений в травні 1925 року. Своє прізвисько автомобіль отримав за назвою фірми, яка випускала біжутерію. На Type 35A був встановлений надійний двигун, запозичений у Type 30. У нього було три підшипника, зменшені клапана, котушки запалювання, і був він менш потужний, ніж його рідний брат Type 35. У загальній складності було вироблено 139 автомобілів Type 35A.

Хоча Етторе Бугатті волів двигуни без наддуву, на Type 35C був встановлений суперчарджер Рутса, який збільшив потужність двигуна до вражаючих 128 кінських сил. Всього було випущено п'ятдесят автомобілів цієї модифікації, однак, багато з них здобули історичні перемоги для компанії. Автомобіль Type 35C виграв Гран-прі Франції в 1928 і 1930 роках, що, безсумнівно, стало найбільшим досягненням команди.

Учасники перегонів

Автомобіль Bugatti Type 35T, відомий як Targa Florio, був спеціально підготовлений для гонки Targa Florio. Всього було випущено лише тринадцять таких автомобілів. Він був оснащений 2, 3-літровим двигуном. Після прийняття змін до правил участі в перегонах серії Гран-прі, встановили допустимий об'єм двигуна менше двох літрів, Type 35T був виведений з експлуатації, а його виробництво було припинено.

Модифікація Bugatti Type 35B була представлена в 1927 році. Це була остання конфігурація лінійки Type 35. Автомобіль Type 35ТС отримав своє ім'я завдяки тому, що був оснащений 2, 3-літровим двигуном, як і Type 35T, а також суперчарджером, як Type 35C. Його двигун видавав приголомшливу потужність у 138 кінських сил, перевершуючи, таким чином, всі інші моделі Type 35. Всього було побудовано 45 таких машин, одним з найбільших досягнень якої стала перемога в Гран-прі Франції 1929 року.

Разом Type 35B почалося виробництво Type 39, однак, цей автомобіль був пристосований до нових гоночним правилами чемпіонату Гран-прі, які обмежували об'єм двигуна 1, 5 літрами. Всього було випущено десять автомобілів Type 39.
