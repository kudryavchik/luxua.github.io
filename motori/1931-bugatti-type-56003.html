Автомобілі Bugatti Type 50, які випускалися в період з 1930 по 1933 року, по суті, були зменшеною спортивного купе версією моделі Type 46 Royale. Автомобіль оснащувався коробкою передач в блоці з задньою віссю і великим п'ятилітровим восьмициліндровим двигуном, який, можливо, був створений на основі гоночного двигуна автомобіля Miller. Type 50, дуже розкішний автомобіль за будь-якими мірками, повинен був замінити модель Type 46.

Дана модель пропонувалася на двох колісних базах і з декількома різними стилями кузова. Він продавався в невеликих кількостях з-за його високої ціни. Автомобіль, розрахований для автомобілістів, які захоплюються спортивними гонками, був розроблений, як високоякісний кар для повсякденної їзди. Команда Bugatti Type 50 вперше брала участь у перегонах у Ле-Мані в 1931 році і виступала тут протягом наступних трьох років. У 1931 році Type 50 деякий час очолював гонку, перш ніж зійшов з траси. В залежності від типу кузова Type 50 міг розвинути швидкість до 169 км/год і розганявся до 100 км/ч менше, ніж за 8 секунд, значна цифра для спортивного автомобіля 30-х років.

У 1931 році Етторе Бугатті представив модель Type 50. Ця модель була створена на базі Type 46, запозичивши багато його механічні компоненти і спираючись на його дизайн. На відміну від Type 46 нова модель відрізнялася восьмициліндровим двигуном з подвійним верхнім розподільним валом, який змінив агрегат з одним верхнім розподільним валом. Це означало, що Type 50 став першим автомобілем Bugatti, який був оснащений ГРМ з двома верхнерасположенными розподільними валами, який до того ж був самим потужним двигуном, спроектованим на заводі Bugatti. Хоча двигун був менше за розміром у зв'язку із зменшенням діаметра циліндра та ходу поршня порівняно з Type 46, він був здатний виробляти 225 кінських сил, хоча в цьому йому допомагав нагнітач Рутса і подвійний карбюратор Zenith. Залежно від стилю кузова автомобіль міг розвинути максимальну швидкість до 169 км/ч.

Type 50 був оснащений традиційною сталевий сходової рамою шасі і був доступний в двох варіантах: на короткій та довгій колісній базі. Type 50Т, маркування Т означала Tourisme або Touring, збирався на довгій колісній базі, і його двигун видавав 200 кінських сил. З його триступеневої механічною коробкою передач і провідними осями він був придатний для тривалих поїздок. Як це було характерно для того часу, компанія Bugatti постачала рухливі шасі різним компаніям, які займалися виробництвом автомобільних кузовів, для того, щоб їх транспортні засоби можна було оформити у відповідності з побажаннями замовника. Це означає, що специфікації автомобілів значно різняться. Більшість транспортних засобів отримали закриті кузова купе.

Жан Бугатті переконав свого батька дозволити трьом автомобілям Type 50 взяти участь у виснажливій гонці «24 години Ле-Мана» в 1931 році. Після того, як в одного з автомобілів почалися проблеми з покришками, інші Type 50 були зняті з перегонів. Автомобілі Type 50 продовжували брати участь у перегонах у Ле-Мані протягом наступних трьох років. Найбільшого успіху їм вдалося досягти у 1935 році, коли на якийсь час їм вдалося очолити гонку.

Виробництво автомобілів моделі Type 50 тривало з 1931 по 1933 рік. Всього було зібрано 65 машин. Type 50 з його дорогим шасі і ретельно розробленим кузовом був призначений для заможних клієнтів. Враховуючи невелику кількість вироблених автомобілів і різні стилі автомобільних кузовів, Type 50 високо цінується як колекційний і рідкісний автомобіль.
