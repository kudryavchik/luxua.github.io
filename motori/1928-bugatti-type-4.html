У 1926 році компанія Bugatti представила оновлений варіант Type 35, Type 35B, з великим восьмициліндровим 2, 3-літровим двигуном з наддувом. Цей двигун був також використаний в їх нової дорожньої машини, Type 43.

Він був оснащений великими гальмами і збільшеним радіатором, який також встановлювався на Type 38, для того, щоб зробити Type 43 більш підходящим для використання на автошляхах. Враховуючи його гоночне походження, було цілком природним для власників і навіть для заводу виставляти Type 43 на такі змагання, як гонки по пересіченій місцевості Brooklands. Серед його відмітних особливостей були дюралюмінієві колеса в стилі Grand Prix. Всього 160 автомобілів Type 43 було зібрано на заводі Bugatti.

  • Boattail Roadster
  • Кузов: Pritchard & Demollin
  • Номер шасі: 43222

Даний автомобіль з шасі номер 43222 володіє унікальним двомісним кузовом від компанії Pritchard & Demollin. Всього кілька автомобілів Bugatti отримали кузова, створені цієї невеликої бельгійською компанією.
