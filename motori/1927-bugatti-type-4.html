Перший Bugatti Type 43 був доставлений покупцеві 16 червня 1927 року. Серцем моделі Type 43 був його 2, 3-літровий рядний восьмициліндровий двигун з наддувом.

За винятком більш низького коефіцієнта стиснення, восьмициліндровий двигун Type 43 з циліндрами, розміщеними в двох блоках по чотири циліндра, був ідентичний двигуна, що встановлюється на гоночний автомобіль Type 35B. Коробка швидкостей автомобіля, осі і радіатор були взяті у Type 38.

Вражаючі показники автомобіля були підкреслені Ф. У. Бредлі, біографом Етторе Бугатті, коли він у супроводі Бугатті здійснив поїздку на одній з ранніх моделей Type 43. Повідомлялося, що при стандартному передаточному числі автомобіль зможе розігнатися до 180 км/ч.

Рама Type 43 була абсолютно новою і була схожа на конструкції на раму гоночного Type 35B. На Type 43 встановлювалися збільшені гальма і радіатор з тим, щоб зробити його більш відповідним для дороги. Bugatti Type 43A, представлений в 1929 році, був більш розкішним і, отже, важче. Враховуючи багату гоночну родовід, було цілком природно, що компанія Bugatti виставляла Type 43 і 43A на такі гонки, як автомобільні гонки по пересіченій місцевості Brooklands в 30-ті років.
