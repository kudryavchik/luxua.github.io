1928 Bugatti Type 37A - це 1, 5-літровий чотирициліндровий Type 37A з наддубом належав легендарному гонщику команди Bugatti П'єру Вейрону, переможцю гонки в Ле-Мане і герою французького руху опору, який також дав своє ім'я нинішньому супер-кару Bugatti.

  • Boattail Racer
  • Номер шасі: 37316

Type 37 був запущений у виробництво в 1926 році, використовуючи злегка змінений двигун Type 40, і відразу ж став одним з претендентів на перемогу на трасах чемпіонату Grand Prix. Компанія Bugatti продала багато безнаддувных автомобілів любителям спортивної їзди, але саме Type 37A з наддувом став однією з найбільш успішних моделей. З 290 побудованих автомобілів Type 37, 67 були оснащені наддувом і отримали маркування Type 37A. Звук працюючого двигуна з наддувом був описаний як «звук розривається перкалю». Для того щоб відрізнити одну модифікацію від іншої, автомобілі Type 37 часто оснащувалися колесами з дротяними спицями, а 37A - дисками з алюмінієвого сплаву.

Type 37 був вперше представлений в 1926 році і випускається в двох видах - звичайному і з наддувом. Версії з наддувом отримали назву 37A. Виробництво тривало до 1930 року. За час виробництва було випущено близько 290 автомобілів. Машина була оснащена дволітровим двигуном з чотирма циліндрами, з карбюратором Zenith або Solex. З допомогою нагнітача двигун видавав 90 кінських сил. Максимальна швидкість дорівнювала майже 160 км/год. Потужність спрямовувалася на задні колеса за допомогою чотириступінчастою механічної коробки передач.

Bugatti Type 35 був оснащений потужним двигуном і, таким чином, був дуже успішним на гоночних трасах. Етторе Бугатті хотів почати масове виробництво Type 35, але порахував, що двигун буде занадто потужним для його використання на дорогах. Тому він взяв двигун від Type 40 і встановив його на шасі Type 35, в результаті чого вийшла модель Type 37. Її версія з наддувом отримала назву Type 37A.
